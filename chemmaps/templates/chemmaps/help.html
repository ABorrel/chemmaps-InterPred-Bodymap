Navigation

Inspired by popular navigation tools such as Google Maps , users have different options accessible from the main panel
(A) A dedicated search bar allowing users to rapidly identify a specific drug based on its chemical name, ID, or generic name;
(B) A description panel including chemical properties such as logP or molecular weight and the actual chemical structure of the selected molecule. This panel also includes options to extract and/or connect up to twenty most similar molecules in that space;
(C) A visualization panel including options to choose which types of drugs are displayed (e.g., approved, withdrawn, in development), and options to setup the color of compounds based on chemical properties (e.g., molecular weight, logP).

Material

DrugMap is solely focusing on the chemical space defined by all drugs from the DrugBank database with 8,752 compounds (release dated 2017-12-20). Only small molecules were considered as peptides and non-organic compounds were removed.


Methods

ChemMaps uses a compendium of 1D, 2D and 3D pre-computed molecular descriptors to generate the drug space in three dimensions. The first two dimensions were defined using a principal component analysis from a set of 648 1D/2D RDKit descriptors computed using the Python library PyDPI. Only informative descriptors (i.e., no null variance, no correlated descriptors with pairwise R> 0.8) are conserved. The third dimension was computed using 432 3D PADEL descriptors filtered using the same protocol. Importantly, 3D chemical structures were generated from SMILES strings using Ligprep from the Schrödinger software suite (release 2017.3), with only the most stable conformation conserved for each chemical.
Drugs’ first and second coordinates were computed from 116 1D and 2D descriptors, representing 14.0% (X-axis) and 8.7% (Y-axis) of the overall variance in the descriptor space. The third coordinate was computed from 70 3D descriptors defining 26.0% of descriptor variability.
